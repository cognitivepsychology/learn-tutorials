Step 1: Set up the Grid

The first step to making any Plotly graph is setting up the data in a grid. You can type your data directly into a grid, or copy it from a spreadsheet and paste it in.

Open a NEW grid from the file well.

Type directly into the cells, or type ctrl-v to paste in clipboard content.

Name your new grid and save it to the filewell, to have it on-hand for future plots and to share it.

You can also import .csv, .xls, or .xlsx files directly into Plotly.

Upload a file using the IMPORT tab.

If you upload a workbook, a new grid will be created for every sheet within the book...

...they’re all stored in a folder named after your file in the filewell.

“View Data”

You can extract the data from any Plotly graph, and build your own graph out of it!

Go to any graph page. The one pictured to the right is https://plot.ly/~slkent/51

Click the VIEW DATA button, located in the lefthand sidebar.

This will duplicate the grid and open it up in your workspace...

...the column names should let you know what role they played in the original plot.

Figure out what data you want in your new graph and either paste it into a new grid or add to this one.

Name your new grid and save it to the filewell, to have it on-hand for future plots and to share it.

Step 2: Plot

Once your grid is set up, you’re ready to plot!

Select your plot type from the MAKE A PLOT menu.

Click the OPTIONS you’d like to enable in the sidebar...

...then set your variables with the buttons that have appeared below the column headers...

...different colors indicate different data sets. These will be plotted as different colors by default.

Click the PLOT button at the bottom of the sidebar to make your plot.

You can either make a new plot (default) or add to an existing one.
